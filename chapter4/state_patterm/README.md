# Stateパターン

組み込み系で重宝するのが、このStateパターン。組み込み系ではハードウェアの状態に応じて動作をするプログラムを書く必要があるので、状態に応じて何らかの分岐処理を実現する必要がある。

ここでは簡単なCDプレーヤを考えてみる。


| ボタン          | 機能            |
|:---------------:|:---------------:|
| [Play or Pause] | 再生 / 一時停止 |
| [Stop]          | 停止            |
